# 📝 Simple To-Do List

A clean, beginner-friendly web-based To-Do List built using **HTML**, **CSS**, and **JavaScript**. This project allows users to add tasks with optional due dates, mark them as completed or pending, and categorize them accordingly — with data persistence using localStorage.

---

## 🔧 Features

- ✅ Add tasks with optional due date
- ✅ Display tasks under "All To-Do Items"
- ✅ Automatically sort tasks into:
  - **Completed**
  - **Pending**
- ✅ Maintain task state even after page reload using `localStorage`
- ✅ Fully responsive and easy to use


---

## 💡 Future Enhancements

* Edit and delete tasks
* Task filtering and search
* Mobile-friendly UI improvements
